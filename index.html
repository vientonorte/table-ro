¿Cómo funciona la extracción automática de BuJo con IA? ▼
1. Ve a 📓 Bullet Journal → sube una foto o escribe texto manualmente.
2. Pulsa Analizar y revisar → . La IA extrae los ítems automáticamente.
3. Revisa, selecciona los que quieres y pulsa ＋ Agregar al tablero .

Configura tu API Key en ⚙️ Admin → Motor IA. Sin key, el texto manual se procesa igualmente.
¿Por qué sale Error 400 al conectar Google? ▼
Google OAuth no permite file:// como origen. Debes servir el tablero desde localhost:

1. Terminal → navega a la carpeta
2. Ejecuta: python3 -m http.server 8080
3. Abre http://localhost:8080/index.html
4. Agrega http://localhost:8080 en Authorized JS Origins ↗
¿Cómo sincronizar Sura Investments (Outlook)? ▼
Opción A: Suscribe el calendario Outlook dentro de Google Calendar.

Opción B: En 🔄 Sync → ⚙️ junto a Sura → pega la URL ICS directamente. Requiere admin habilitado.
¿Los cambios se sincronizan automáticamente? ▼
El sync es manual: usa 🔄 Sync para traer eventos, 💾 Guardar para persistir el estado local, y 📤 Enviar a Calendarios para empujar todas las tarjetas locales a Google Calendar según su categoría.
