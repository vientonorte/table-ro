¿Cómo funciona la extracción automática de BuJo con IA? ▼
1. Ve a 📓 Bullet Journal → sube una foto de tu BuJo físico.
2. Ingresa tu Anthropic API Key en ⚙️ Admin → BuJo IA.
3. Haz clic en 🤖 Analizar con IA — Claude Vision analiza la imagen directamente.
4. El resultado se parsea automáticamente en tarjetas con categorías.
5. Selecciona las que quieres y haz clic en ＋ Agregar al tablero.

Sin API Key: puedes pegar el texto manualmente en el área de texto.
¿Por qué sale Error 400 al conectar Google? ▼
Google OAuth no permite file:// como origen. Debes servir el tablero desde localhost:

1. Terminal → navega a la carpeta
2. Ejecuta: python3 -m http.server 8080
3. Abre http://localhost:8080/index.html
4. Agrega http://localhost:8080 en Authorized JS Origins ↗
¿Cómo sincronizar Sura Investments (Outlook)? ▼
Opción A (recomendada): Agrega el calendario Outlook como suscripción en Google Calendar → Settings → "Add calendar" → "From URL" → pega la URL ICS de Outlook. Una vez suscrito, se detecta automáticamente con 🔍 Detectar cals.

Opción B: En 🔄 Sync → ⚙️ junto a Sura → pega la URL ICS directamente. Requiere admin habilitado en la pestaña Permisos.
¿Qué significan los permisos de calendario? ▼
R/W (Lectura+Escritura) → Personal: puedes traer eventos Y empujar tarjetas manuales a Google Calendar con el botón 📅.

Solo lectura → Finanzas: solo se importan eventos, no se puede escribir.

Admin requerido → Sura Investments: requiere habilitación explícita en ⚙️ Permisos por razones de seguridad corporativa.

Consulta compartida → Camila: requiere que ella haya compartido su calendario contigo en Google Calendar.
¿Los cambios se sincronizan automáticamente? ▼
No. El sync es manual: usa 🔄 Sync para traer eventos, y 💾 Guardar para persistir el estado local. Los eventos solo se envían a Google Calendar cuando usas el botón 📅 en una tarjeta específica (solo si estás conectado con OAuth).
