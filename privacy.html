Qué es TABLERO Rö

TABLERO Rö es una aplicación web personal de planificación semanal que integra Google Calendar, feeds ICS, Trello (opcional) y análisis de Bullet Journal con IA multimodal. Es de uso personal y no comercial.

Responsable del tratamiento

Rö Gaete Gaona — contacto: gaete.gaona@gmail.com

Autenticación con Passkey

TABLERO Rö puede proteger el acceso mediante passkeys (WebAuthn) :

  • Autenticación biométrica (Face ID, Touch ID, Windows Hello) o PIN del dispositivo.
  • Credenciales en hardware seguro (TPM, Secure Enclave) — no contraseñas.
  • Vinculación al dominio — resistente a phishing.

Datos biométricos: nunca abandonan tu dispositivo; la app no accede a huella ni rostro.

Almacenamiento local

La mayoría de los datos viven en tu navegador . No operamos base de datos propia ni vendemos datos.

Procesamiento en servidor (proxy)

Si configuras un Cloudflare Worker como proxy:

  • Las solicitudes de IA pasan por ese servidor; las API keys residen allí, no en tu navegador (v1.5+ en producción).
  • Las imágenes BuJo se envían al proveedor IA solo para extracción; no se almacenan en disco del Worker ni en GitHub Pages. La lectura crómática Clave A (v1.6) clasifica por color de marcador en el cliente tras la respuesta.
  • Las URLs ICS pueden proxearse; el Worker solo reenvía a hosts permitidos.

Terceros

  • Google — Calendar API y OAuth.
  • Anthropic / OpenAI / Google AI — solo si usas análisis BuJo vía proxy o dev local.
  • Trello / Atlassian — solo si configuras sync Espacio Seguro.
  • GitHub Pages — hosting estático (sin acceso a datos locales).

Seguridad

  • WebAuthn/Passkeys — FIDO2 sin contraseñas.
  • API keys IA — no se persisten en producción; usar proxy Worker.
  • CSP y sanitización — mitigación XSS en contenido de usuario.
  • HTTPS — conexiones cifradas.

Eliminación de datos

  • Cerrar sesión passkey: ⚙️ Admin → 🔐 Seguridad · Passkey → Cerrar sesión
  • Eliminar passkeys: ⚙️ Admin → Gestionar passkeys
  • Limpiar tablero: ⚙️ Admin → Limpiar estado guardado
